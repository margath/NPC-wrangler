🛠️ Setup Instructions for the DM

To use this tracker, you need to link it to a Google Sheet using Google Apps Script.

  1. Set Up the Google Sheet:
    • Create a new Google Sheet.
    • Add two tabs at the bottom. Name the first one NPCs and the second one PCs.
    • In the PCs sheet, type PlayerName, CharacterName, and PIN as column headers to create player logins.
  2. Add the Code:
    • Click Extensions > Apps Script.
    • Paste the provided database script below (replacing any existing code).
    • Click the Save icon.
  3. Deploy the Web App:
    • In the top right corner, click Deploy > New deployment.
    • Click the gear icon ⚙️ next to "Select type" and choose Web app.
    • Under "Execute as", select Me.
    • Under "Who has access", select Anyone.
    • Click Deploy and authorize permissions.
  4. Link the App:
    • Copy the Web app URL provided by Google.
    • Paste it into the input box above!
